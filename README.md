# Outcomee

Outcomee is a bilingual React + TypeScript MVP for a decision intelligence product concept. The app now includes a polished marketing layer plus a working in-browser product flow for account access, story publishing, story discovery, and a personal dashboard.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Stack

- React
- TypeScript
- Vite

## Notes

- Language switching is handled client-side.
- Accounts, sessions, and stories are stored in browser localStorage for a fully runnable local MVP.
- Seeded demo accounts are available:
	- `selin@outcomee.app` / `demo123`
	- `mert@outcomee.app` / `demo123`
- New users can also sign up directly from the app and immediately publish stories.
